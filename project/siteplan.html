eBite Site Plan

Site Name

The name "eBite" was inspired by "eBay" because it is a fictional site where you can sell "used" and "refurbished" fast food rather than electronics.

Site Purpose

The site is a shopping site where users can buy and resell fast food.

Scenarios

  • Where can I buy a half-eaten cheeseburger?
  • Where can I sell my half-eaten cheeseburger?
